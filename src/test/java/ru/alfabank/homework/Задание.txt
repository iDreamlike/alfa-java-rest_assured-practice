Работаем с сервисом https://jsonplaceholder.typicode.com/
Под каждый блок запросов создаем отдельный класс в тестах.
Для работы с телом запроса и ответами создаем классы с описанием нужных полей
В ответе базово проверяем код и выводим содержимое ответа.

- GET-запросы(тут дополнительно проверяем что в ответе содержатся
              все необходимые поля - в двух вариантах - через объекты и с помощью валидации схемы json в rest assured):
    1.Получение списка всех пользователей.
    2.Получение информации о конкретном пользователе.
    3.Получение списка всех постов.
    4.Получение информации о конкретном посте.
    5.Получение списка всех комментариев.

- GET-запросы с разными параметрами:
    1.Получение списка постов конкретного пользователя.
    2.Получение списка комментариев конкретного поста.
    3.Получение списка альбомов конкретного пользователя.
    4.Получение списка фотографий из конкретного альбома.
    5.Получение списка дел (todos) конкретного пользователя.

- POST-запросы:
    1.Создание нового пользователя.
    2.Создание нового поста.
    3.Создание нового комментария.
    4.Создание нового альбома.
    5.Создание новой задачи (todo).

- DELETE-запросы:
    1.Удаление пользователя.
    2.Удаление поста.
    3.Удаление комментария.
